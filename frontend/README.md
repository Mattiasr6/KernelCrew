CodeCore - Frontend (Angular SPA)Este es el cliente de la plataforma CodeCore, desarrollado en Angular 21. Se comunica con una API REST de Laravel para la gestión de cursos, usuarios y pagos.🚀 Sprint 1: Estado ActualSe ha completado la arquitectura base y los módulos de identidad y gestión.Autenticación: Flujo completo con Laravel Sanctum (Tokens).Gestión de Usuarios: CRUD administrativo con protección de rutas.Gestión de Cursos: Catálogo público y paneles privados para Admin e Instructores.🛠️ Configuración del ProyectoRequisitos PreviosNode.js: v18 o superior.Angular CLI: v21.2.7.InstalaciónEntra en la carpeta del frontend:Bashcd frontend
Instala las dependencias (incluye Angular Material):Bashnpm install
Inicia el servidor de desarrollo:Bashng serve
La aplicación estará disponible en http://localhost:4200/.🔌 Conexión con el Backend (Laravel)Para que el frontend funcione correctamente con la API, asegúrate de:Tener el servidor de Laravel corriendo en http://localhost:8000.Habilitar CORS en Laravel para permitir peticiones desde el puerto 4200.Configurar el archivo src/environments/environment.ts si la URL de la API es distinta.Endpoints Esperados por el FrontendMóduloEndpointMétodoAuth/api/login, /api/register, /api/logoutPOSTUsuarios/api/admin/usersGET, POST, PUT, DELETECursos/api/coursesGET, POST📂 Estructura del Códigosrc/app/core/: Contiene los interceptors para el token de Sanctum, los guards de roles y los servicios globales de API.src/app/features/: Módulos funcionales (Auth, Admin, Courses, My-Courses).src/app/shared/: Componentes reutilizables y utilidades.src/app/core/models/: Interfaces TypeScript. Backend Dev: Por favor, consulta estos archivos para ver el formato JSON esperado.🧪 Comandos ÚtilesTests: ng test (Usando Vitest).Build: ng build (Genera la carpeta /dist).💡 Nota para el Desarrollador de BackendLas interfaces de datos están definidas siguiendo el README principal del proyecto. Si realizas cambios en los nombres de las columnas de la base de datos que afecten al JSON de salida, por favor notifícalo para actualizar los modelos de TypeScript.
